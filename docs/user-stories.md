# User Stories

# SmartDesk AI Platform

**Version:** 1.0

---

# US-001 - User Login

**As a** registered user,

**I want** to log into the platform,

**So that** I can securely access my dashboard and support tickets.

### Priority

High

### Acceptance Criteria

* User can log in with valid credentials.
* Invalid credentials display an error message.
* User is redirected to the dashboard after successful login.

---

# US-002 - Create Support Ticket

**As an** end user,

**I want** to create a support ticket,

**So that** I can request technical assistance.

### Priority

High

### Acceptance Criteria

* User can create a new ticket.
* Ticket receives a unique identifier.
* Ticket status is set to "Open".

---

# US-003 - Assign Ticket

**As a** support agent,

**I want** to assign tickets to technicians,

**So that** requests are handled by the appropriate person.

### Priority

Medium

### Acceptance Criteria

* Agent can assign a technician.
* Assignment history is recorded.

---

# US-004 - Resolve Ticket

**As a** support agent,

**I want** to close resolved tickets,

**So that** completed requests are properly finalized.

### Priority

Medium

### Acceptance Criteria

* Ticket status changes to "Resolved".
* Resolution date is stored.

---

# US-005 - AI Assistant

**As an** end user,

**I want** to interact with an AI assistant,

**So that** I can solve common issues more quickly.

### Priority

High

### Acceptance Criteria

* AI provides suggested solutions.
* User can request human assistance if necessary.

---

# US-006 - Dashboard

**As an** administrator,

**I want** to view support metrics,

**So that** I can monitor team performance.

### Priority

Medium

### Acceptance Criteria

* Dashboard displays KPIs.
* Reports are updated in real time.

---

# US-007 - Manage Users

**As an** administrator,

**I want** to manage user accounts,

**So that** I can control system access.

### Priority

High

### Acceptance Criteria

* Administrator can create users.
* Administrator can edit users.
* Administrator can deactivate users.
